# ncm decoder

# How to use
```shell
多线程异步ncm音频解码器

Usage: ncm2mp3.exe [OPTIONS] --ncm-dir <NCM_DIR>

Options:
  -n, --ncm-dir <NCM_DIR>
          ncm所在目录

  -o, --out-dir <OUT_DIR>
          文件输出目录

          [default: unlocked]

  -t, --threads <THREADS>
          线程数量

          [default: 64]

  -h, --help
          Print help (see a summary with '-h')

  -V, --version
          Print version
```
